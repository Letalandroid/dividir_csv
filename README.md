# Dividir CSV
Script sencillo que divide el archivo en 3 partes y mantiene el encabezado en cada uno

## 🔧 Cómo usarlo:
1. Guarda este código como dividir_csv.py en la misma carpeta donde está tu .csv.
2. Abre una terminal o cmd en esa carpeta.

3. Ejecuta:
```ruby
python dividir_csv.py
```

4. Obtendrás parte1.csv, parte2.csv, parte3.csv en la misma carpeta.
